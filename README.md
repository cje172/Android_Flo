## Description 📋
* **Android & Layout**
	* 이론 : 프론트 엔드 개발자, Kotlin 언어, Layout, View에 대해 설명한다.
	* 프로젝트 환경설정 실습 : 프로젝트 진행 전 Android Studio 프로젝트 환경설정에 대해 다룬다.
	* 간단 xml 파일 실습 : xml 파일을 제작하며 화면을 만드는 방법에 대해서 다룬다.
	* 클론코딩 실습 : 홈 화면 만들기 FLO 클론 코딩을 진행한다.
 * **화면 구성 및 기능 구현**
	* 이론 : Activity & Fragment 에 대해 설명한다.
	* Activity 화면전환 : Activity 전환 방법에 대해 배우고 View Binding을 사용하여 정적인 화면에 생명력을 부여한다.
	* Fragment 화면전환 : Fragment 전환을 하는 방법에 대해서 배운다.
	* 화면 간 데이터 전달 : data Class 와 Intent 를 사용하여 Activity 간 데이터 전달에 대해서 배운다.
 * **유용한 위젯들**
	* 이론 : 유용한 위젯들 (Tab Layout, ViewPager2) 에 대해 설명한다.
	* ViewPager 사용법 : ViewPager를 어떻게 사용하는지에 대해 배운다.
	* Tab Layout 사용법 : ViewPager를 Tab Layout과 함께 사용한다.
	* Bottom Navigation 리뷰 : 템플릿에 있는 Bottom Navigation이 어떤 흐름으로 구성되어 있는지 배운다.
 * **스레드 (Thread)**
	* 이론 : 스레드(Thread)에 대해 설명한다.
	* 실습 : 스레드를 사용해 Seek Bar 노래 재생을 구현한다.
 * **생명주기**
	* 이론 : 생명주기 이론 설명 및 이해를 위한 간단한 프로젝트를 실행한다.
	* 실습 : 생명주기를 사용해 노래 Status 변경 및 노래 데이터를 저장한다.
 * **리스트 뷰 (List or Recycler View)**
	* 이론 : 리스트 뷰(리사이클러 뷰)에 대해 설명한다.
	* 실습 : 리사이클러뷰로 오늘 발매한 음악 페이지를 구현한다.
 * **Room DB**
	* 이론 : 안드로이드 Local 에서 사용할 수 있는 DB에 대해 설명한다.
	* 실습 : Room DB를 이용해 지금까지의 Song 데이터를 Song List 로 전환하고 다음 노래 재생, 좋아요 기능을 구현한다.
 * **네트워크 API**
	* 이론 : 네트워크 및 API에 대해서 설명한다.
	* 실습 : 서버와의 통신을 구현한다.
